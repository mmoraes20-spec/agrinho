## 📋 Sobre o Projeto

**Agro Forte, Futuro Sustentável** é uma landing page educacional desenvolvida para conscientizar sobre a importância da sustentabilidade no agronegócio. O projeto apresenta de forma clara e visualmente atraente como é possível conciliar alta produtividade no campo com a preservação dos recursos naturais.

O site aborda temas como:
- 🌳 Preservação ambiental e proteção da biodiversidade
- 💧 Uso consciente da água e irrigação inteligente
- 🚜 Tecnologia sustentável e agricultura de precisão
